이 시뮬레이션은 무엇인가요?

이 시뮬레이션은 건물 사이 골목에 설치된 "스마트 센서 게이트"가 지나가는 사람을 감지하는 과정을 보여줍니다. 카메라(신장 추정), 열감지 배열(체온 분포), 마이크(심박·호흡), 입자 센서(휘발성 유기화합물)까지 — 실제 배선을 연결해야만 각 센서의 데이터가 흐릅니다.

  1. 각 센서 케이블(CAM/HEAT/MIC/PART)을 아래 아두이노 핀에 연결하세요.
  2. AUTO / TOP / GATE / SIDE 버튼으로 카메라 각도를 바꿔가며 사람과 센서 반응을 관찰하세요.
  3. 사람은 계속 지나다닙니다 — 연결되지 않은 센서는 항상 NO SIGNAL 입니다.
  4. 모든 배선과 모든 카메라 각도를 다 살펴보면, 숨겨진 버튼이 나타납니다.
